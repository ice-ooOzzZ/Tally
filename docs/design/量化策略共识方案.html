1讨论机制设计（怎么达成一致）

三位研究员被刻意设定为互相冲突的投资世界观，保证分歧是真实的而非表演：

研究员 A · 趋势动量派强者恒强，价格包含一切信息；赚市场反应不足的钱；认为"接飞刀"是散户亏钱主因
研究员 B · 基本面质量派股价长期由盈利驱动；赚定价滞后与错杀折价的钱；认为纯技术信号是噪音交易
研究员 C · 均值回归 + 风控派短期过度反应普遍存在；首席怀疑官：多数策略死于风控缺失而非信号错误
① 独立提案三人互不可见
各提 3 条，共 9 条
② 交叉质询逐条攻击他人提案
预判并回应对自己的攻击
③ 投票各交一份"最终5条"名单
允许合并、允许砍自己的
④ 主席裁决共识直接采纳
分歧按票数+论据强度裁决

达成一致的四条元规则（讨论中自发形成，值得复用）：

  1. 不裁决世界观，只裁决可证伪性：接飞刀好还是追涨好不辩论，每条策略必须自带"什么情况下算被证伪、就废弃"的验收条件（参数敏感性军令状）。
  2. 重叠策略按"选股归基本面、择时归价格"分工合并：谁的条件更不易过拟合保留谁的。
  3. 尾部风险必须显式定价：不允许"靠扛"；每条策略要算清最坏情形对组合的损失（仓位上限 × 单票最大亏损）。
  4. 择时权统一上收：各策略删除自带的指数级闸门，市场状态判断由唯一的组合级闸门负责，避免重复参数和双重绞杀。

策略〇 · 市场状态闸门

基础设施3:0 通过来源：C3 + B 的长短仓分离修正 + A 的降仓规则补全

作用：用基准指数数据把市场分为牛/震荡/熊三态，作为五条策略的总开关与仓位调节阀。它不赚钱，它省掉熊市里接飞刀和动量崩溃的钱。A股/美股各用各的基准独立判态。

状态判定（每日收盘，T+1 生效）

trend = 指数close > SMA(指数close, 200)
mom   = 指数close / 指数close[t-60] − 1          # 60日动量
vol   = std(指数日收益, 20) × √252                # 20日年化波动
dd    = 指数close / max(指数close, 近250日) − 1   # 距一年高点回撤

熊市   = (非trend 且 mom < −5%) 或 dd < −20%     # dd是独立熔断条款
牛市   = trend 且 mom ≥ 3% 且 vol ≤ 25%（A股vol阈值放宽至30%，写死禁改）
震荡市 = 其余
防抖：状态切换需连续 5 个交易日满足新状态才正式切换

对五条策略的调制

状态短持类（策略一/二/四）长持类（策略三/五）总仓上限
牛市正常运行正常运行80%
震荡正常运行（策略四的最佳环境）正常运行60%
熊市禁止新开仓；存量时间止损减半禁止新开仓；存量按自身规则自然退出（B 的修正：不强制腰斩长持仓）短持类合计≤25%

切熊降仓顺序（A 要求补全）：超出上限时按浮亏从大到小、非跌停日限价卖出。

上线验收（可证伪条件）① 防抖天数 3/5/7、mom ±5%、dd −20% 三组参数敏感性矩阵备案，判定结果对参数高度敏感则闸门本身视为过拟合；② 2018 A股、2022 美股回测中，闸门须把挂接策略的最大回撤压到裸信号的 60% 以下，否则回炉。已知成本：V 型反转（2020/3 型）必然滞后，错过反弹第一波，这是买尾部保护的保费。

策略一 · 趋势突破（52周新高 × 量能确认）

趋势 · 右侧3:0 通过来源：A1 + 泡沫刹车 + C 的四项验收条件

逻辑：赚锚定效应的钱——投资者锚定历史高点，股价接近前高时因"太贵"惜买，好消息被消化不足；真突破后压制解除，信息缓慢扩散推动续涨。锚定是人性，不会被套利掉。

数据：日K（价/量/额）
持有：1–4月
单票上限：8%
市场：两者（美股更干净）

买入条件（T 日收盘全部满足，T+1 开盘执行）

A. close[T] ≥ max(high[T-250 .. T-1])            # 突破250日最高
B. volume[T] ≥ 1.5 × mean(volume[T-20 .. T-1])   # 放量确认——核心交叉证据：
                                                  # 价新高×量1.5倍 = 有增量资金背书，缺一即假突破
C. close[T] > SMA(close, 50)                      # 中期趋势冗余保护
D. 泡沫刹车：池内20日新高股票占比 > 40% 时暂停新开仓  # 信号过密本身是过热指标（2015教训）
E. 剔除当日涨幅 > 9% 的A股（次日追高滑点毁灭信号）
F. 并发排序：多个突破信号同日出现时，按 12-1 动量分降序优先（A2 的遗产）
注：原自带指数>SMA200 闸门已删除，择时权上收策略〇

卖出条件（任一触发，T+1 开盘执行）

a. 吊灯止损: close < max(high[入场日..T-1]) − 3×ATR(20)   # 水位用T-1数据，防未来函数
b. 趋势破位: close < SMA(close, 50) 连续 3 日
c. 初始硬止损: close < 入场价 × 0.92
上线验收 + 已知失效场景验收：① 回测按次日开盘价成交并计入 1% 滑点惩罚，惩罚后年化超额转负则废弃（B 的条件）；② 1.5×量能 / 3×ATR / −8% 参数敏感性报告，任一翻转回炉（C 的条件）；③ 涨停突破的两种处理（剔除 vs 次日竞价接力）差异报告。失效场景：震荡市假突破绞杀（胜率可低至 30%，靠盈亏比与闸门生存）；T+1 下跳空穿止损，实际止损分布按 −12% 建模；连续 8–10 次止损是统计必然，评估看期望不看胜率。

策略二 · 事件 PEAD（盈利加速 × 放量确认）

事件驱动3:0 合并通过来源：A3 ⊕ B1（事件锚定归 A，基本面验真归 B）

逻辑：赚财报好消息定价滞后的钱（PEAD，60 年文献最顽固的异象）。基本面腿保证"改善是真的"（防营收注水），价格腿保证"市场投票承认且尚未定价完"（防买到已被业绩预告透支的事件）——这是两派各出一半的完整策略。

数据：季度财务（含披露日）+ 日K
持有：1–2月
信号：财报季月3–6个
市场：两者（对市况依赖最低）

买入条件（事件驱动）

# 事件日 R 的定义写死（C 的要求，防定义腐蚀）：业绩预告与正式披露中信息增量更大的
# 那一次为事件；盘后/非交易日公告 → R 顺延至次一交易日
基本面腿（B 的贡献，字段一律按披露日+1生效）:
  F1. 营收同比增速 > 上一季度营收同比增速，且 > 0    # 增速加速，看边际不看绝对
  F2. ROE_ttm ≥ 8%                                   # 质量底线
  F3. 经营现金流 OCF > 0                              # 现金流验真，防并购/一次性收入注水
价格腿（A 的贡献，核心交叉证据）:
  P1. R日: 涨幅 ≥ +3% 且 volume ≥ 2.0 × 20日均量      # 放量跳涨 = 市场投票"真超预期"
  P2. R 后 5 个交易日内 close ≥ high[R] → 触发买入      # 回踩收复，逐日可判定
  P3. close > SMA(close, 200)
超过 R+5 未触发 → 信号作废不追。F×P 同时成立才是"可持续漂移"的证据：
只有财务好没放量 = 市场不买账；只有跳涨没财务 = 消息炒作。

卖出条件（任一触发）

a. 时间退出: 持有满 45 个交易日（漂移有半衰期；30/45/60 敏感性检验）
b. 基本面证伪: 新财报 营收转负 或 OCF < 0 → 披露日+1 无条件清仓（B 的贡献）
c. 硬止损: −12%（两家折中；A 原"破R日低点"因风险距离过宽被弃用）
d. 下次财报披露前 1 日若浮盈 < 5% → 提前退出（不赌下一次财报）
已知失效场景A股业绩预告制度会稀释正式披露日的信息量——价格腿 P1 正是防线：已被预告透支的披露日不会再放量跳涨；美股大盘股 PEAD 漂移窗口近年缩短，45 日时间退出的敏感性检验为此而设；单季财务可被一次性损益扭曲，F3 与 b 是仅有的两道防线。

策略三 · 质量折价回归（质量 × 估值 × 企稳）

左侧价值3:0 合并通过来源：B2 ⊕ C2（选股归 B，择时与风控归 C）

逻辑：赚优质公司被恐慌错杀的钱——机构因排名压力被迫抛售、ETF 赎回无差别砸盘造成的定价误差，而这个折价在只做多市场没有天然套利者。三张表交叉是灵魂：财务（公司没坏）× 估值（便宜了）× 价格（跌够了且止跌）。任何单一维度都会失败。

数据：季度财务 + 周度快照(PB) + 日K
持有：0.5–1年
信号：月1–3个
市场：两者（A股折价更深、回归更暴力）

买入条件（全部满足）

质量门槛（B 的贡献，防价值陷阱）:
  A1. ROE_ttm ≥ 12%，且连续最近 4 个已披露季度 ROE_ttm ≥ 10%
  A2. 近 4 个已披露季度中至少 3 季 OCF > 0
  A3. 最新营收同比 > −15%（允许轻度承压，排除断崖）
  A4. 剔除：存在未回复交易所问询函 / 年报非标审计意见（B 接受质询后新增的防造假补丁）
估值折价（核心证据）:
  B1. PB ≤ 自身过去 3 年周度序列 20% 分位（BPS 严格按披露日生效——A 质询后明确）
  B2. PE > 0（剔除亏损导致的假低PB）
入场纪律（C 的贡献，替换 B2 原"钝化"条件——该条件被判定为放行慢性阴跌）:
  C1. 距 60 日高点回撤 > 15%，且相对基准 20 日超跌 > 8%   # 确认是"被砸"不是"横着便宜"
  C2. 近 3 日止跌: close > close[t−3]                     # 最低限度的右侧确认
  C3. 财报新鲜度: 距最新披露日 ≤ 60 个交易日               # 不拿一年前的财报当护身符
退出后同一股票 60 个交易日冷却。

卖出条件（任一触发）

a. 估值回归: PB 回升至自身 3 年 60% 分位 → 止盈
b. 基本面证伪: 新财报 ROE_ttm < 8% 或 连续两季 OCF < 0 → 无条件清仓（C 版：更果断）
c. 硬止损: −15%（C 版否决了 B 原来的 −20%）
d. 时间退出: 250 个交易日未触发 a/b → 退出（一年不回归说明折价有你看不到的理由）
已知失效场景价值陷阱是头号杀手——ROE 是滞后指标，"还没垮但商业模式已被判死刑"的公司挡不住，靠 c/d 兜底；高ROE+低PB+现金流正常与部分造假公司画像重合（A4 补丁只能降频不能根除，靠单票上限消化）；系统性熊市中分位持续下移会反复接刀，靠策略〇熊市禁新开仓托底。低频策略样本少，回测结论主动打折（C 的自我警告保留在案）。

策略四 · 恐慌错杀回补（超卖短线）

短线均值回归3:0 通过来源：C1 + B 的基本面健康检查 + C 自加的财报回避窗口

逻辑：赚被迫卖出者的钱——止损盘、赎回、杠杆平仓的抛售不是基于新判断而是"必须卖"；T+1 和涨跌停延缓价格修复，给次日入场留下利润空间。只买长期趋势完好的"活人"，绝不接"死人"的刀。

数据：日K + 最新财报
持有：2–7日
信号：震荡市月4–8个
单票上限：5%

买入条件（全部满足）

活人判定: close > SMA200 且 SMA200 今日值 > 20日前值      # 长期趋势健康
挨刀判定: RSI(3) < 15  且  close < SMA5  且  5日累跌 > 6%  # 短期恐慌超卖
崩溃排除（区分"恐慌"与"出事了"）:
  近5日无跌停（美股等效：无单日−15%）；当日非跌停收盘
  5日均成交额 / 60日均成交额 < 3.0                        # 排除天量出货
健康检查（B 的贡献）: 最新已披露财报 ROE_ttm > 0 且 OCF > 0
事件回避（C 质询后自加）: 距下一已知财报/预告披露日 > 5 个交易日
# 三维交叉证据：趋势数据(200日线) × 短期价格(RSI3+累跌) × 微观结构(跌停/天量排除)

卖出条件（按优先级）

a. 目标达成: RSI(3) > 60 → 修复即走，不贪
b. 时间止损: 持有满 7 个交易日无条件卖出（熊市减半为 3 日——策略〇调制）
c. 灾难止损: close < 入场价 × 0.93
d. 趋势失效: close < SMA200
A股跌停无法成交时次日继续挂卖；5% 仓位上限即为此场景预留（最坏连续跌停单票
−30% × 5% = 组合 −1.5%，尾部已显式定价）。
军令状（上线验收，C 本人立下）① RSI3 阈值 10–20、5日累跌 5%–8% 全区间结果必须同向且期望为正，任一翻转整条废弃；② 补"次日低开>5% 按开盘价止损"的现实撮合规则重跑回测；③ 无法区分恐慌与暴雷第一波是本策略原罪——健康检查与回避窗口只能降频，残余风险由仓位上限定价，不假装不存在。

策略五 · 盈利动量持有（基本面长持）

基本面趋势2:1 主席裁决入选来源：B3 收紧版（C 的三项收紧 + A 的验收门槛 + B 的转轨联动）

逻辑：赚"持续性怀疑折价"的钱——连续多季兑现高增长的公司，市场每季度都重新怀疑一次其持续性，而经营趋势有强惯性。本质是承担"增长突然中断"的尾部风险换取的风险溢价，不是免费的。与策略三买点相反（买强势 vs 买折价），构成风格对冲。

数据：季度财务 + 周度快照(PE) + 日K
持有：2–4季度
信号：月0–2个
同持上限：8只

买入条件（每月末检查）

A. 连续 3 个已披露季度: rev_yoy > 10% 且 ROE_ttm > 15%   # 一季是运气，三季是能力
B. 近 2 季 OCF > 0
C. PE ≤ 自身 3 年 70% 分位            # C 收紧: 原 80 分位约束力≈0，2020年底白马全通过
D. 过去 120 日累计收益 > 基准指数      # 长窗口相对强度=市场持续认可的价格影子，
                                       # 刻意钝化，避免退化成月度价格动量
超额信号按 ROE_ttm 降序取前 8。
联动（B 的贡献）: 策略二持仓 45 日到期时若满足本策略 A–D → 转入长持轨道。

卖出条件

a. 基本面中断（即时）: 新财报 rev_yoy < 0 或 ROE_ttm < 12% 或 OCF < 0 → 退出
   # 增长股的第一次坏消息几乎从不是最后一次，不等第二季确认
b. 移动止损（即时）: 持有期最高收盘价回撤 −20%（C 收紧: 原 −25% 是验尸报告不是止损）
c. 估值泡沫（月末）: PE > 自身 3 年 95% 分位 → 退出
# 原"120日落后基准>15%退出"已删除——被 C 判定为"风格底部割肉机"，风格风险改由策略〇管理
上线验收 + 已知失效场景验收（A 的反对意见转化）：① 回测必须按"跌停日不可卖出"撮合规则重跑，白马雷的一字跌停是本策略固有成本；② 删除 A–D 任一条件后收益大变则视为过拟合废弃。失效场景：2021 式核心资产崩塌是已经演习过的尾部剧本（书面承认在案）；风格切换期连续 6–12 个月跑输是常态，必须以 3 年视角评估；高 ROE 可能由高杠杆驱动（无负债字段），靠行业上限约束。

6.2 与系统规则引擎的映射

  • 每条策略实现为 PRD §5.1 规则引擎中的一个 StrategyRuleSet（一组 Rule 类 + 独立的退出状态机），策略〇实现为组合级 RegimeGate 中间件。
  • 五条策略分别独立回测、独立归因（PRD §6 回测引擎），再做组合级合成回测——单策略不达标不允许进组合。
  • 全部"上线验收"条款（敏感性军令状、现实撮合复测、滑点惩罚）写入 PRD §11 测试策略，作为 M3 决策门的验收清单。
  • 信号追踪（PRD §6.6）按策略分桶统计成绩单，任一策略实盘滚动表现与回测预期严重背离时单独停用，不牵连其余。
